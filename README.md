# sus-crm-kivy
Aula sobre uma tela de CRUD básica em kivy.

# Após a instalação do python, versão 3.12.

# Execute a cadeia de comandos abaixo.

py -3.12 -m venv .venv
.\.venv\Scripts\activate
python -m pip install --upgrade pip
pip install kivy

# Após a instalação do kivy, execute o programa:
python app/main.py.
